# 🧩 Tech Words Crossword

A **mobile-friendly crossword puzzle website** where you can choose from categories like **Python, C++, PHP, JavaScript**, or play with a **mixed set**.  
It dynamically generates a crossword grid with clues, lets you type answers directly, and provides helpful tools like **Check progress** and **Reveal mistakes**.

## ✨ Features
- 🔀 Select category: Python, C++, PHP, JavaScript, or Mix  
- 📱 Fully responsive, works on mobile and desktop  
- 🧠 Dynamic puzzle generation with numbered clues  
- 🎯 Keyboard + tap navigation  
- ✅ Check answers and reveal mistakes  
- 🚀 Works offline, no tracking  

## 🚀 Getting Started
1. Download or clone this repo.  
2. Open `index.html` in your browser.  
3. Choose a category and click **New Puzzle**.  
4. Tap/click cells or use arrow keys to play.
